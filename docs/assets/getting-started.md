# Getting Started with ZerithDB in 5 Minutes

Welcome to ZerithDB 🚀

ZerithDB is a lightweight peer-to-peer database designed for modern web applications with real-time
synchronization.

In this tutorial, you will:

- Install ZerithDB using npm or pnpm
- Create a typed schema
- Insert and query data
- Sync data across browser tabs in real time
- Persist data locally
- Deploy your application to Vercel or Netlify

By the end, you will have a fully working collaborative todo application.

---

# Prerequisites

Before starting, make sure you have:

- Node.js 18+
- npm or pnpm installed
- Basic JavaScript knowledge

Check versions:

```bash
node -v
npm -v
```
